Detailed Description

Subroutine to solve triadiagonal system of equations.

Solves for a vector u of size N the tridiagonal linear set using given by equation 2.4.1 in Numerical recipes in Fortran 90 ([70]) using a serial algorithm. Input vectors b (diagonal elements) and r (right-hand side) have size N, while a and c (off-diagonal elements) are not defined in the first and last elements, respectively.
